Welcome to the VOT Toolkit documentation
========================================

The VOT toolkit is the official evaluation tool for the [Visual Object Tracking (VOT) challenge](http://votchallenge.net/). 
It is written in Python 3 language.

Contributions and development
--------------------

Contributions to the VOT toolkit are welcome, the preferred way to do it is by submitting an issue or a pull requests on `GitHub <https://github.com/votchallenge/toolkit>`_.
